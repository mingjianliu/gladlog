### Task 2: 时间戳解析 + 顶层 CSV 切分

**Files:**

- Create: `packages/parser/src/l1/timestamp.ts`、`packages/parser/src/l1/splitTopLevel.ts`
- Test: `packages/parser/test/l1.tokenize.test.ts`

**Interfaces:**

- Produces:
  - `parseTimestamp(datePart: string, opts?: { timezone?: string }): number | null` — 输入形如 `6/30/2026 19:10:31.7312`(月/日/年,毫秒 4 位),返回 epoch ms。无 timezone 时按系统本地时区。非法输入 null。
  - `splitTopLevel(s: string): string[]` — 按顶层逗号切分:双引号内逗号不切(引号剥离,`\"` 转义处理);`[]`/`()` 内逗号不切(括号保留在参数字符串里)。
  - `splitLine(line: string): { datePart: string; eventName: string; params: string[] } | null` — 整行入口:日志行 = `日期 时间(两个空格)EVENT,params...`;返回三段;非法返回 null。

- [ ] **Step 1: 写失败测试(完整契约)**

```ts
// packages/parser/test/l1.tokenize.test.ts
import { parseTimestamp } from "../src/l1/timestamp";
import { splitTopLevel } from "../src/l1/splitTopLevel";
import { splitLine } from "../src/l1/splitTopLevel";

describe("parseTimestamp", () => {
  it("parses current-format timestamp with year to epoch ms (UTC given)", () => {
    const ms = parseTimestamp("6/30/2026 19:10:31.7312", { timezone: "UTC" });
    // 2026-06-30T19:10:31.731Z
    expect(ms).toBe(Date.UTC(2026, 5, 30, 19, 10, 31, 731));
  });
  it("respects timezone offset", () => {
    const utc = parseTimestamp("6/30/2026 19:10:31.0000", { timezone: "UTC" })!;
    const berlin = parseTimestamp("6/30/2026 19:10:31.0000", {
      timezone: "Europe/Berlin",
    })!;
    expect(utc - berlin).toBe(2 * 3600 * 1000); // 6 月柏林 = UTC+2
  });
  it("returns null on garbage", () => {
    expect(parseTimestamp("not a date")).toBeNull();
    expect(
      parseTimestamp("13/45/2026 99:99:99.0000", { timezone: "UTC" }),
    ).toBeNull();
  });
});

describe("splitTopLevel", () => {
  it("splits plain params", () => {
    expect(splitTopLevel("1825,41,3v3,1")).toEqual(["1825", "41", "3v3", "1"]);
  });
  it("keeps commas inside double quotes and strips the quotes", () => {
    expect(splitTopLevel('123,"Name-With,Comma",0x512')).toEqual([
      "123",
      "Name-With,Comma",
      "0x512",
    ]);
  });
  it("keeps commas inside [] and () nesting, brackets preserved", () => {
    expect(splitTopLevel("71,[(90269,112121,1),(90270,112122,1)],9")).toEqual([
      "71",
      "[(90269,112121,1),(90270,112122,1)]",
      "9",
    ]);
  });
  it("handles nested [[...],[...]]", () => {
    expect(splitTopLevel("[[1,2],[3,4]],x")).toEqual(["[[1,2],[3,4]]", "x"]);
  });
  it("empty params become empty strings", () => {
    expect(splitTopLevel("a,,b")).toEqual(["a", "", "b"]);
  });
});

describe("splitLine", () => {
  it("splits a real line into date/event/params", () => {
    const r = splitLine(
      "6/30/2026 19:10:31.7312  ARENA_MATCH_START,1825,41,3v3,1",
    )!;
    expect(r.datePart).toBe("6/30/2026 19:10:31.7312");
    expect(r.eventName).toBe("ARENA_MATCH_START");
    expect(r.params).toEqual(["1825", "41", "3v3", "1"]);
  });
  it("returns null when there is no double-space separator or no comma", () => {
    expect(splitLine("garbage line without structure")).toBeNull();
    expect(splitLine("")).toBeNull();
  });
});
```

### Task 3: 事件族 decoder(基础三元组 + spell/damage/heal/aura/interrupt/absorb/died/arena)

**Files:**

- Create: `packages/parser/src/l1/decoders.ts`(或按族拆多文件,实现者定,导出面见下)
- Test: `packages/parser/test/l1.decoders.test.ts`

**Interfaces:**

- Produces(供 Task 4 的 parseLine 派发):
  - `decodeBaseUnits(params: string[]): { srcGuid, srcName, srcFlags, srcRaidFlags, destGuid, destName, destFlags, destRaidFlags }`(params[0..7];flags 为 number,由 `0x...` 解析;`nil` 名字 → null)
  - `decodeSpell(params, at)`: `{ spellId: number, spellName: string, spellSchool: number }`(SPELL*\* 事件 params[8..10];SWING*\* 无此段)
  - `decodeDamage(tailParams)`: `{ amount, baseAmount, overkill, school, resisted, blocked, absorbed, critical: boolean, effectiveAmount }`;effective = amount − max(overkill, 0)
  - `decodeHeal(tailParams)`: `{ amount, baseAmount, overheal, absorbed, critical: boolean, effectiveAmount }`;effective = amount − overheal(下限 0)
  - `decodeAdvanced(params, at)`: `{ actorGuid, ownerGuid, hp, maxHp, x, y, facing, mapId }`(damage/heal/cast_success 的 advanced 载荷段,17 个参数)
  - `decodeAura(tailParams)`: `{ auraType: 'BUFF' | 'DEBUFF', amount?: number }`
  - `decodeExtraSpell(tailParams)`: `{ extraSpellId, extraSpellName, extraSchool }`(INTERRUPT/DISPEL/STOLEN)
  - `decodeAbsorbed(params)`: `{ shieldOwnerGuid, shieldOwnerName, shieldSpellId, shieldSpellName, absorbedAmount, totalAmount, critical: boolean }`(SPELL_ABSORBED 特殊结构:攻击方三元组+受击方三元组+攻击 spell 段+盾主三元组+盾 spell 段+数额)
  - `decodeArenaStart(params)`: `{ zoneId, unkInstanceId, bracket, isRated }`;`decodeArenaEnd(params)`: `{ winningTeamId, matchDurationSeconds, team0Mmr, team1Mmr }`
